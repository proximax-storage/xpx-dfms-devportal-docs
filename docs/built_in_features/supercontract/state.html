SuperContract State

Any SC can have the following states:

NameMeaning
Unknownthe SC status is unknown (execution lifecycle does not imply this state. If a user gets it that means something is wrong)
Startedthe SC is executing
Successthe SC execution finished successfully
Failurethe SC execution finished failure
